# Online Store

#### Prepare project:

```sh
1) download project from gitHub
2) download h2DB
3) download Tomcat10
```

#### How run project:

```sh
1) mvn clean install
2) run h2DB and use command: java -jar h2_fileName.jar
3) run OnlineStore_SU-1.0-SNAPSHOT.war on Tomcat 10
4) open in browser http://localhost:8080/
```
